# Segment-anything-4DFlow

This repository contains data, models, and supplementary materials for our work on applying the Segment Anything Model (SAM) and its variants to 4D flow MR scans and digital subtraction angiography (DSA) images.

## 📚 Publications

If you find the resources in this repository helpful, please consider reading and citing our papers:

**Authors:** Amirkhosro Kazemi, A Ghazipour, T Settle, MF Stoddard, Amir A Amini
